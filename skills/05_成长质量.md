## 1. 收现比验证（v2.0 新增核心逻辑）

### 收现比定义与计算
```
收现比 = 销售商品提供劳务收到的现金 / 营业收入
```

### 收现比三档判定（快速上涨标的必须通过此关）
| 收现比 | 判定 | 成长类型 |
|--------|------|---------|
| > 1.1  | 🏆 **预收主导** | 产业链话语权极强（茅台/华为供应商模式） |
| 0.8-1.1 | ✅ 正常 | 现金流健康，成长可持续 |
| 0.6-0.8 | ⚠️ 信用扩张 | 赊销比例高，需关注坏账风险 |
| < 0.6 | 🔴 **现金流预警** | 判定为"虚假成长"，降评至D级 |

### 收现比趋势审计
- **收现比持续下降**（连续2年）：应收账款可能爆雷，发出"成长陷阱预警"
- **应收账款增速 > 营收增速 × 1.5**：系统触发强制预警，判定为信用泡沫
- **A股特有**：季报中的"销售商品收到现金"比年报更具时效性

## 2. 营收含金量审计

### 快速上涨标的的现金流双验证
对于近期涨幅较大的标的，必须执行以下双验证：
1. **OCF/净利润比值**：< 0.6 → 判定涨幅由博弈驱动，非成长驱动
2. **收现比**：< 0.8 → 判定营收质量存疑，可能是游资题材炒作

**结论逻辑**：
```
IF 收现比 < 0.8 AND OCF/净利润 < 0.6:
    → 判定为"游资博弈驱动"，不属于成长驱动上涨
    → 成长评级降至 C 或 D
ELSE IF 收现比 > 1.0 AND OCF/净利润 > 0.8:
    → 判定为"现金流驱动的高质量成长"
    → 成长评级可达 A+
```

## 3. 规模效应与经营杠杆

### 边际利润贡献测试
- 营收每增长 1%，归母净利润需增长 > 1.2% → 存在经营杠杆（优质成长）
- 营收增长但净利润增长 < 0.5% → "增收不增利"，判定为规模陷阱

### 毛利率背离审计（一票否决项）
- 营收增长 > 15% 但毛利率下降 > 3% → 判定"赔本赚吆喝"
- 成长评级**直接降为 D**，并在报告中明确标注

### A股特有成长陷阱识别
| 陷阱类型 | 识别特征 | 应对 |
|---------|---------|------|
| 政府补贴型 | 补贴/净利润 > 30% | 扣非后PEG重新计算 |
| 关联交易型 | 关联收入占比 > 20% | 质疑营收真实性 |
| 商誉堆砌型 | 并购频繁 + 商誉/净资产 > 30% | 关注业绩承诺到期 |
| 应收膨胀型 | 应收/营收 > 30% 且持续上升 | 收现比降级处理 |

## 4. 成长驱动力分类

最终必须将标的归类为以下四种之一：

- **💎 创新驱动**：技术/产品/商业模式形成护城河 → 最高估值溢价
- **📈 规模驱动**：市场份额扩张 + 规模效应降本 → 中等估值溢价
- **💰 价格驱动**：主要靠提价推动营收 → 低估值溢价
- **🎭 虚假扩张**：收现比差 + 毛利率下降 → 无估值溢价，判定为博弈标的

---

## 5. PEG 与天花板估值

### PEG 计算（A股专版）
```
PEG = 当前PE(TTM) ÷ 预期净利润增速（%）
```
- PEG < 0.8：被低估的高质量成长（价值+成长双击）
- PEG 1-1.5：合理区间
- PEG > 2.5：成长溢价过高，除非是竞争格局极好的龙头
