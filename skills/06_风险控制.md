## 1. ATR 波动率极值审计（v2.0 新增核心逻辑）

### ATR(14) 历史分位数判断
**从数据上下文读取 `atr_analysis` 字段：**
- `atr14`：14日平均真实波幅（绝对值）
- `atr_pct`：ATR 占股价的百分比
- `atr_percentile`：当前 ATR 处于历史的百分位数
- `volatility_label`：引擎预判标签

### ATR 四档风险判定
| ATR历史分位 | 波动状态 | 风险含义 | 操作建议 |
|------------|---------|---------|---------|
| < 25%分位  | 🟢 **历史低波动** | 压缩待发，突破时爆发力强 | 布局窗口 |
| 25-60%分位 | 🟡 正常波动 | 趋势运行健康 | 正常持仓 |
| 60-80%分位 | 🟠 偏高波动 | 注意止损纪律 | 减仓止盈 |
| > 80%分位  | 🔴 **历史高波动** | 极端行情，风险敞口大 | 触发一票否决 |

### ATR 与回撤关联分析
对于"快速上涨且回撤小"的标的：
- ATR处于低位 + 价格稳步上涨 → 主力控盘，低回撤有结构性支撑
- ATR处于高位 + 价格快速拉升 → 游资拉抬，随时可能高位震荡出货

**一票否决触发条件：**
```
IF ATR历史分位 > 85% AND 近5日涨幅 > 15%:
    → 判定为"高波动快速拉升"，强制评级不超过"中性持有"
    → 必须在报告中发出"高位波动风险警告"
```

## 2. 股权质押"生死线"审计

### 爆仓风险计算公式
```python
平仓价 = 质押时股价 × (1 - 质押率) × 0.85
生存空间 = (当前股价 - 平仓价) / 当前股价 × 100%
```
- **生存空间 < 15%**：判定为"二级高危"，强制建议规避
- **质押率 > 50%**：触发一票否决，总评不得超过"中性"

## 3. A股灰犀牛红旗清单（一票否决项）

以下任一指标触发即执行一票否决，不可被其他专家覆盖：

| 红旗指标 | 阈值 | 否决理由 |
|---------|------|---------|
| 大股东质押率 | > 50% | 强制平仓风险随时引发踩踏 |
| 商誉/净资产 | > 35% | 减值黑洞，利润随时清零 |
| 经营现金流 | 连续2年为负 | 不能自我造血，靠融资续命 |
| 审计意见 | 非标准 | 财务真实性存疑 |
| 近12月问询函 | > 2份 | 监管重点关注对象 |
| ATR历史分位 | > 85% + 近5日涨幅 > 15% | 高位高波，出货概率极大 |
| 收现比 | < 0.5 | 营收真实性存疑（与05专家联动） |

### 大存大贷审计
- 货币资金充裕但短期借款同样高居不下
- 逻辑矛盾：若有钱，为何借款？→ 资金被大股东占用嫌疑

## 4. 波动率 × 成长质量联动审计

从其他专家报告获取：
- **05_成长质量**：若成长评级 < C，且ATR高位，双重叠加风险加倍
- **04_资金博弈**：若筹码状态为"大规模派发"，且ATR高位，判定为出货尾声

```
综合风险矩阵：
  ATR高位 + 成长差 + 筹码派发 → 极高危（100分），强制一票否决卖出
  ATR正常 + 成长好 + 筹码锁仓 → 低风险（20分），可积极配置
```

## 5. 安全垫分析与止损策略

### 基于ATR的动态止损计算
```
止损位 = 当前价格 - (ATR14 × 2)
建议持仓风险 = ATR14 × 2 / 当前价格 × 100%
```
- 单笔持仓风险 > 8% 时，建议仓位不超过总资金的 15%
